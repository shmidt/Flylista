Flylista
========

Flylista is a flights schedules for norwegian airports,
based on AvinorAPI.

iOS6 and ARC required.

It's my second app, so please don't expect much :)

App at App Store: [https://itunes.apple.com/app/flylista/id486934512?mt=8](http://)
